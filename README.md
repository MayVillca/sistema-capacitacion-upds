# 🎓 Sistema de Gestión Académica — Centro de Capacitación

Sistema web desarrollado en Laravel para la administración académica de un centro de capacitación.  
Permite gestionar cursos, docentes, estudiantes, inscripciones, asistencia y calificaciones,  
con cálculo automático del estado final (Aprobado/Reprobado) por estudiante.
